## 🎯 Project Overview

Viridian Football is an ambitious NFL General Manager simulation game designed to be the most authentic, deep, and strategically complex representation of the GM role ever created. The project leverages AI-assisted development to build a web-based simulation with unprecedented realism.

### **Key Differentiators**
- **USE Engine**: Advanced simulation with body states, spatial awareness, fatigue modeling, and injury risk
- **AI GM Archetypes**: Personality-driven AI with distinct decision-making patterns
- **Fog of War**: Realistic scouting uncertainty and information gaps
- **Emergent Storytelling**: Dynamic narratives driven by player relationships and team chemistry
- **Modding Support**: Community-driven content creation and customization

### **Technology Stack**
- **Core Engine**: Java with WebAssembly compilation
- **Frontend**: React with TypeScript
- **Database**: PostgreSQL with Redis caching
- **API**: REST API with WebSocket support
- **Platform**: Web-based with future desktop potential
